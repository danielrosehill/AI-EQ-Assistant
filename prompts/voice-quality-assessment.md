# Voice Quality Assessment

## Prompt

```
Listen to the voice sample I've provided and assess the overall voice quality.

Please provide:

1. **GRBAS Scale Assessment**
   Rate each parameter from 0 (normal) to 3 (severe):

   | Parameter | Score (0-3) | Notes |
   |-----------|-------------|-------|
   | **G**rade (overall severity) | | |
   | **R**oughness (irregular vibration) | | |
   | **B**reathiness (air leakage) | | |
   | **A**sthenia (weakness) | | |
   | **S**train (hyperfunctional) | | |

2. **MOS-Style Ratings**
   Rate each on a scale of 1-5 (1=poor, 5=excellent):

   | Quality Aspect | Rating (1-5) | Notes |
   |----------------|--------------|-------|
   | Overall Quality | | |
   | Listening Effort Required | | |
   | Naturalness | | |
   | Comprehension/Clarity | | |
   | Pleasantness | | |

3. **Voice Characteristics**
   - Timbre description (warm, bright, thin, rich, nasal, etc.)
   - Resonance quality
   - Vocal clarity
   - Any signs of vocal fatigue or strain

4. **Technical Quality**
   - Recording quality (room noise, artifacts, etc.)
   - Microphone technique observations
   - Signal-to-noise assessment

5. **Best Use Cases**
   Based on the voice characteristics, suggest ideal applications:
   - Podcast/conversational
   - Audiobook narration
   - Commercial voiceover
   - Documentary narration
   - Character work
   - Other recommendations

6. **Overall Assessment**
   - Strengths of this voice
   - Areas for potential improvement
   - Overall rating out of 10

Provide honest, constructive feedback that would help the speaker improve or understand their voice's characteristics.
```

## What This Analyzes

- **Clinical voice quality (GRBAS)**: Standard perceptual voice assessment scale
- **Perceptual quality (MOS)**: Mean Opinion Score style subjective ratings
- **Voice timbre**: Tonal characteristics and color
- **Technical quality**: Recording and microphone factors
- **Suitability**: What content types suit this voice

| Score | Description |
|-------|-------------|
| 0 | Normal |
| 1 | Slight deviation |

| 3 | Severe deviation |
